# Security Policy

## Supported Versions

This project is currently pre-1.0. Security fixes will target the latest published version.

## Reporting a Vulnerability

Please report security issues privately through GitHub Security Advisories when available.

Do not open a public issue for vulnerabilities that expose sensitive data, authentication flaws, or executable payloads.

When reporting, include:

- affected version or commit,
- steps to reproduce,
- expected and actual behavior,
- suggested fix if you have one.
