# Changelog

All notable changes to this starter kit will be documented here.

## 0.1.0 - Unreleased

### Added

- Blade-first Laravel starter kit structure.
- Tailwind CSS 4 and daisyUI 5 setup in `resources/css/app.css`.
- Theme switching through `data-theme`, daisyUI `theme-controller`, and localStorage persistence.
- Blade UI wrappers under `resources/views/components/ui`.
- Component playground at `/playground`.
- Generated daisyUI catalog backed by `config/daisyui.php`.
- Documentation-style examples, props, and usage snippets for playground pages.
- Feature tests for the starter surface, handwritten playground pages, and generated daisyUI routes.

### Notes

- This is a starter repository first. Composer package extraction should happen after the Blade component API stabilizes through real usage.
