# Code of Conduct

Be respectful, constructive, and patient.

This project is meant to help Laravel developers build faster with Blade and daisyUI. Keep discussion focused on improving the starter, its component APIs, documentation, tests, and release quality.

Unwelcome behavior includes harassment, personal attacks, discrimination, and intentionally disruptive comments.

Project maintainers may remove comments, issues, or pull requests that do not follow this standard.
